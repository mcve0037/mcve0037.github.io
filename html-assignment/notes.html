HTML Tags

Headings - "h1, h2, h3, h4, h5, h6"
This tag is used for headings on a page. The number after the "h" corresponds to the importance of the text, "1" being the most important.
Blockquote - "blockquote"
This tag is used as a large, stand alone quote.
Citation - "cite"
This tag is used to cite another source, it's often used with blockquotes/quotations.
Abbreviation - "abbr"
This tag is used to show abbreviations/acronyms in a expanded form. Example: IMD
Important Text - "strong"
This tag is used to strongly emphasize important text. The text will become bolded.
